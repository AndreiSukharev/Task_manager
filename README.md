# Debate app
App for managing task


[Server] Go

[Client] Vue

[Database] PostgreSQL

[Deployment] Docker


## Getting Started

#### Install go, npm
```
brew install go npm
```

#### Install docker

```
https://docs.docker.com/compose/install/
```

## Build and Run

```
git clone https://github.com/AndreiSukharev/debate.git task
cd task
go build main.go
cd client
npm install
npm run dev
front: http://localhost:8080
server: http://localhost:5010/ 
```
